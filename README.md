# Adversarial-ML-MNIST
Adversarial ML project: FGSM attack on MNIST CNN with clean vs adversarial accuracy analysis and visual results.
## Overview
This project demonstrates how adversarial attacks can fool deep learning models.
A CNN model is trained on the MNIST dataset, and FGSM (Fast Gradient Sign Method)
is used to generate adversarial samples that reduce model accuracy.

## Objectives
- Train a CNN on MNIST
- Implement FGSM attack
- Evaluate model performance on clean vs adversarial images
- Visualize adversarial examples

## Technique Used
- Dataset: MNIST
- Model: CNN
- Attack: FGSM (Fast Gradient Sign Method)
